# To run this project


## Install Cobra CLI
```
go install github.com/spf13/cobra-cli@latest
```
> [!NOTE]
> Execute all the following inside your active directory
 

## Build your project
```
go build
``` 

## Install your project into the build
```
go install
```

## To run this application
```
cli
```

## To check the add function

```
cli add 2 3 4 5 
```
#### With  flags
```
cli add 2 3 4 5 -f
```
